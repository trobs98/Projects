double Math::evaluate_expression ( const std::vector< token *> &  expr)
staticprivate

Evaluates an array of tokens to a number.

It assumes that the only operations are +, -, *, and /, which means that it can be evaluated as a sum/difference of products/quotients. It also assumes that the user won't give an expression that is meaningless (ie. "Evaluate 1 over 0", or "Evaluate 1 + + 2").
